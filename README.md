
##### wedding_hotel_sys



##### 命令

运行项目:

$ cd myApp
$ ionic serve

本地地址 http://localhost:8100/



打包成单页面项目运行在微信/web浏览器:

$ ionic build



打包成混合app项目:

$ ionic build
$  ionic cordova platform add ios
$  ionic cordova run android
